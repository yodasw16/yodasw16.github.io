Hi, I’m Dave!

My name is Dave Gillhespy. I am a Senior Web UI Engineer at Quicken Loans, but my made up title doesn't really matter. What matters is that I absolutely love HTML and CSS. It's way more than just a job for me. I am enamored by responsive web design, semantic markup, and progressive enhancement. I often dabble in javascript and try to be a designer, but HTML and CSS are my true loves.

Outside of work, when I'm not doing side projects, I love to write, I love to read, I love music, and I love movies.

What I'm Proud Of

  • Scales – A design-free, modular collection of OOCSS patterns, framework settings, and utilities for quickly building responsive web projects.
  • MyQL.com – a mobile-first, responsive, mortgage origination website (web app, whatever). Launched in February 2012. As far as I know, the first of its kind.
  • Stellar – a live music photography and review site that I built and write reviews for.
  • To Flex or Not To Flex – a calculator for determining if flex fuel is more affordable than regular gasoline. Node API and AngularJS application by Calvin Bushor.
  • Devtroit – a badge and website for designers and developers to show pride in Detroit. Inspired by Boston Built.

Tools of the Trade

The most important tools that I use are Sass, Object Oriented CSS, and Responsive Web Design. These have all changed the way I think about and build websites. Additionally, I never stop fighting for the old standbys: user experience, accessibility, performance, progressive enhancement, and semantic markup.
